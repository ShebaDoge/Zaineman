Tutorials

How to make canards:

Start off with a small rotator. Set DisableBaseMesh to true (optional w XML editor). turn the speed to 100% and change the input to TAS > 20 ? (AngleOfAttack/35)-(Pitch/2) : 0 Make sure to add a wing on the end of it.

How to make single-engine thrust vectoring:

Start off with a hinge rotator. Duplicate it and place it horizontally of the first one. Move the engine away and place the hinges. Put the engine back on. On both of them set the speed to 100% and the range of motion to 20°. Change the front hinge input to pitch. Change the back hinge input to Yaw without invert. Somtimes the angle may vary depending on weight and size. you can turn part cliping off and move them back together

How to make twin-engine thrust vectoring:

copy the same process as before except only change the the left front rotators input to Pitch-Roll and the right front rotators to Pitch+Roll. you might have to switch the lines of core and or invert them for it to work.
